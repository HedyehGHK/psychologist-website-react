# Psychologist Clinic Website (React Version)

This project is a React-based version of a psychologist clinic website.
It was originally built using HTML, CSS, and JavaScript, and later refactored into a modern React application to improve structure, reusability, and scalability.

## 🚀 Project Overview

The website is designed to present information about a psychologist, including services, background, and contact details, in a clean and user-friendly interface.

## 🛠️ Technologies Used

* React (Vite)
* JavaScript (ES6+)
* HTML5
* CSS3

## ✨ Features

* Component-based architecture
* Responsive layout
* Clean and modern UI design
* Reusable components (Navbar, sections, layout)
* Single-page application structure

## 📁 Project Structure

* `src/components` – reusable UI components
* `src/App.jsx` – main application layout
* `src/assets` – images and static files

## 🔄 Project Purpose

This project was created as a learning exercise to:

* Practice React fundamentals
* Convert a static multi-page website into a React application
* Understand component-based design
* Improve front-end development skills

## 📌 Future Improvements

* Add routing for multiple pages
* Implement a functional contact form
* Integrate backend services
* Enhance UI/UX design

## ▶️ Getting Started

To run the project locally:

```bash
npm install
npm run dev
```

Then open:

```
http://localhost:5173
```
